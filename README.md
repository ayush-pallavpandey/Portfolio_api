# Portfolio API

A Spring Boot REST API project demonstrating backend development concepts.

## Features

- GET APIs
- POST APIs
- PUT APIs
- DELETE APIs
- RequestBody
- PathVariable
- RequestParam
- Service Layer Architecture
- Global Exception Handling
- Swagger/OpenAPI Documentation
- Maven Build & Packaging

## Tech Stack

- Java 17
- Spring Boot
- Maven
- Swagger OpenAPI
- Git
- GitHub
